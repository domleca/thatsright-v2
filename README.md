# That's Right — V2

The V2 redesign of [thatsright.xyz](https://www.thatsright.xyz). A daily
voice presence for someone whose memory has begun to waver.

This is a single-page static site. No build step. Edit `index.html`, hard
refresh, ship.

## Structure

- `index.html` — the English page. All CSS, copy (EN + FR), and behavior
  scripts are inline.
- `fr/index.html` — the French page, generated by `../generate_fr.py`
  from `index.html` so Google can index French independently. Do not edit
  by hand — re-run the generator after touching French copy in `index.html`.
- `i18n.js` — the runtime language switcher; navigates between `/` and `/fr`
  on click and forces FR boot when the URL path starts with `/fr`.
- `orb.js` — the canvas orb animation engine, loaded by `index.html`.
- `privacy.html` — the standalone privacy policy page, English only for now,
  linked from the footer. Vercel serves it at `/privacy` (via `cleanUrls`).
- `sitemap.xml`, `robots.txt`, `llms.txt`, `og.png` — SEO + share assets.
- `vercel.json` — `cleanUrls` + the privacy 301 + cache headers.
- `DESIGN_HANDOFF.md` — the original design brief from the designer
  (kept for reference; describes the V2 visual system in detail).
- `TODO.md` — open items still needed before this is fully production-ready.

## Local preview

From the repo root:

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000/`. Hard-refresh after edits (⌘+Shift+R)
so CSS/JS isn't cached.

## French copy

French translations live in the `COPY` dict inside `index.html` (alongside
the English ones) and are also baked into static HTML at `fr/index.html`
so Google can index the French version independently.

To update the French copy:

1. Edit the `fr:` block of the `COPY` object in `index.html`.
2. From the repo root, run `python3 ../generate_fr.py` (or just
   `python3 generate_fr.py` from the project root one level up).
3. Hard-refresh both `/` and `/fr/` locally and verify the change.
4. Commit `index.html` and the regenerated `fr/index.html` together.

The language switcher (top right) navigates between `/` (EN) and `/fr`
(FR) — clicking it triggers a full page load so each URL stays canonical
for search engines. The `FR_ENABLED` flag at the top of `i18n.js` hides
the switcher and forces English if you ever need to roll French back.

## Deployment
