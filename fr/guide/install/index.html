Installer l'application

  1. Sur l'iPhone de votre parent, ouvrez le lien d'invitation TestFlight que nous vous avons envoyé.
  2. Si TestFlight n'est pas encore installé, l'App Store s'ouvre pour vous permettre de l'obtenir. Installez TestFlight, puis revenez au lien d'invitation.
  3. Touchez Installer, puis Ouvrir.
