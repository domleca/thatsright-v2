Installer l'application

  1. Sur l'iPhone de votre parent, ouvrez le lien d'invitation que nous vous avons envoyé.
  2. Si TestFlight n'est pas encore installé, l'App Store s'ouvre pour vous permettre de l'obtenir, puis revenez au lien.
  3. Touchez Installer, puis Ouvrir.
Vous devriez voir L'icône That's Right sur l'écran d'accueil, et l'application s'ouvre sur un écran de bienvenue.
